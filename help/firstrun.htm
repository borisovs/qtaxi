Первый запуск

При первом запуске, программа не знает куда вы установили сервер баз данных и его параметры.

Вам необходимо настроить эти параметры.
Для этого нажимаете "OK" и в окне которое покажет программа ввести:
1. dns имя или ip адрес сервера
2. Имя рабочей базы данных
3. Имя администратора сервера баз данных
4. Пароль администратора сервера баз данных

После этого перезапускаете программу, чтобы изменения вступили в силу.
Первый запуск происходит от имени пользователя "Admin" c паролем "admin" - без кавычек!

Этот пользователь существует только в базе Qtaxi и к администратору сервера не имеет никакого отношения.
