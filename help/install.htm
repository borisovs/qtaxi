Установка программы

Windows
Скачайте с сайта программы файл Qtaxi_setup.exe и запустите его на выполнение.

Выбираете язык установки.

После приветствия программы, жмёте "Далее".
Затем, если вы принимаете условия лицензии GNU GPL v2, отмечете это чекбоксом и опять нажимаете "Далее".

Выбираете компоненты установки программы.
Здесь содержится как пакет самой программы, так и необходимые библиотеки для запуска программы, собранной компилятором VC++ 2005 Express SP1.

После установки программы, происходит установка runtime библиотек в операционную систему. Необходимо нажать "Yes".

На этом установка закончена. В меню "Пуск" и на рабочем столе будут созданы ярлыки для запуска и удаления программы.

Linux
Скачайте с сайта программы пакетный файл для вашего дистрибутива или исходные коды для самостоятельной компиляции.
В случае пакетного файла, воспользуйтесь пакетным менеджером вашего дистрибутива.
Если вы предпочитаете самостоятельную компиляцию, распакуйте архив с исходными текстами в директорию и в терминале наберите комманды.

Компиляция возможна, только при установке библиотек qt4-devel-4.3.1 или выше.

После компиляции, можете запускать программу коммандой ./Qtaxi
